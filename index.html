Relationship Between Covid Deaths, Covid Vaccinations, and Continent from 2020-2022¶

Introduction¶

The objective is to find a correlation between covid cases, covid vaccines, and the location of the continent between 2020 and 2022 and come up with an overall conclusion about the relationship. We will start with analyzing monthly covid deaths and vaccinations and see if there is astring relationship between the location, deaths, and vaccinations.

2. Data Wrangling¶

We see that this dated is organized by day. Since it may be easier to analyze aggregate data, we want to use monthly covid data instead. So we'll need to revise the dataframe by ordering everything by month. Since this is 3 years worth of data, we can enumerate months from 1(January 2020) to 36(December 2022) to make it easier to read when graphed.

In [157]:
# use hashes to reorganize the data into consecutive months
yearsHash = {}
yearsHash[2020] = 0
yearsHash[2021] = 12
yearsHash[2022] = 24

# separate out the year and month to make it easier to handle

global_covid_df['month'] = pd.DatetimeIndex(global_covid_df['date']).month

# we create a new column for the consecutive number of months
global_covid_df['monthByYear'] = 0
for index, row in global_covid_df.iterrows():
    global_covid_df.at[index,'monthByYear']= row['month']+yearsHash[row['year']]

We now want to find some sort of correlation between covid vaccinations and covid deaths since the graphs above show each continents' vaccinations and deaths spiked or increased around the same time. I created 6 plots for each continent and measured the correlation individually. I tried to find a regression line for each continent to better visualize the correlation.

# initializes the plots for each continent
fig, ax =plt.subplots(6, figsize=(10, 20))

# linear regression function to find the regression line for each plot more conveniently
def linearRegressionFunc(df):
    X = np.array(df['new_vaccinations']).reshape(-1, 1)
    y = np.array(df['new_deaths']).reshape(-1, 1)
    regr = LinearRegression()
    regr.fit(X, y)
    y_pred = regr.predict(X)
    #returns the X values, y values, predicted y values, coefficient, and intercept of regression line
    return (X, y, y_pred, regr.coef_, regr.intercept_)

Hypothesis Testing¶

Now that we've visually established a correlation, let's try to prove it via a hypothesis test.

Null Hypothesis: There is no correlation between total_vaccinations_per_hundred, new_deaths_smoothed_per_million, and continent of covid deaths.

To test this, we first try to find the coefficients based off total_vaccinations_per_hundred, new_deaths_smoothed_per_million, and continent of covid deaths and fit a regression line using this data.

Conclusion¶

We see that all p-values are less that 0.05 , which is the most commonly used significant value. This means that we can reject the null hypothesis, that the months between 2020-2022, continent, and total vaccinations per hundred have no correlation with new deaths smoothed per million, and that they in fact do have a relationship.

In conclusion, location and vaccination rates due correlate with covid deaths. This could imply that the continent you live in may influence how strong the correlation is. For example, we see the correlation in North America and South America to be more positive than other continents. This could be because governement restrictions or political views slow the rate vaccines can reach citizens. Additionally, poverty and transportation could influence this correlation. But we do see vaccinations, for sure, correlated negatively with covid deaths, which likely mean they help prevent them.

In the future, further research can be done using this data by analyzing factors that influence the positive correlation between some locations and covid deaths.
